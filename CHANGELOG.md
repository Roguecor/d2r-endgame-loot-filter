# Changelog

## General Changes (v1.3)

* Added enumeration format `"XX. "` to all rules
* Removed word `"Show"` from rule names
* Gold threshold adjusted:

  * `10000 → 5000`
* Removed rare eth/soc **Crystal Sword** rule (merged into misc rares & bases)
* Removed redundant **Jewels** entry from `.09`
* Renamed:

  * `Rare Accessories` → **Rare Rings and Amulets**
  * `Potions` → **Rejuvenation Potions**
* Added:

  * **Super Potions** rule (disabled by default)
* Disabled:

  * **Rare All Class Equipment**
* Consolidated:

  * Rare Boots/Gloves/Belts into broader high-value rules
  * Magic/Rare Circlets into High Value Magic/Rare categories
